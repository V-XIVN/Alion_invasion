# 游戏：外星人入侵
---------------
## 语言：python
* 使用 ___ pygame第三方库 ___ 进行开发  
    * 在运行该项目之前请确保您安装了这样的库  
---------------
## 游戏规则
* 玩家通过左右和空格控制底部的飞船移动和射击  
- 外星人从顶部向下移动，并随着波次速度提高  
* 当外星人撞到玩家飞船或者到达屏幕底部，玩家则会损失一艘飞船  
- 玩家一共有三艘飞船，全部损失则游戏失败
